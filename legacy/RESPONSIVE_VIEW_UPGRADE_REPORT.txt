UPKK SmartKids v2 - Responsive View Upgrade v2.10

Tujuan:
- Menstabilkan paparan untuk PC, laptop, tablet, Android dan iPhone.
- Mengekalkan konsep mobile app/PWA tetapi memberi ruang lebih baik pada skrin besar.
- Mengurangkan masalah layout sempit, bottom nav bertindih dan iOS input zoom.

Fail dikemaskini:
- style.css
- index.html
- app.html
- admin.html
- admin-login.html
- repair-admin-login.html
- reset-firebase.html
- manifest.json
- sw.js

Perubahan utama:
1. Responsive shell:
   - Telefon: kekal seperti app mobile.
   - Tablet: lebar hingga 760px dengan grid 2 kolum.
   - Laptop/PC: lebar hingga 1180px/1220px dengan grid 12 kolum.

2. Bottom navigation:
   - Telefon: kekal fixed bawah.
   - Tablet/desktop: lebar ikut shell, lebih kemas dan tidak terlalu sempit.
   - Desktop: ikon + teks sebaris untuk lebih profesional.

3. App content:
   - Kad, latihan, exam, statistik dan grid disusun ikut saiz skrin.
   - Hero/section penting kekal full-width.
   - Text overflow dan image overflow dikawal.

4. iPhone/Android:
   - Font input 16px untuk elak auto zoom.
   - Gunakan 100dvh untuk viewport moden.
   - Safe-area inset dikekalkan untuk iPhone notch.

5. Admin panel:
   - Admin/login/reset pages ditambah responsive override.
   - Table admin lebih mudah dibaca pada skrin kecil.

6. PWA update:
   - sw.js cache name dinaikkan kepada upkk-smartkids-responsive-2.10.
   - manifest start_url dinaikkan kepada v2.10.
   - orientation ditukar kepada any supaya tablet/laptop lebih fleksibel.

Nota deploy:
- Upload semua fail dalam ZIP ini.
- Jika user masih nampak layout lama, refresh sekali atau tutup/buka semula app PWA.
- Tidak perlu import JSON Firebase semula.
