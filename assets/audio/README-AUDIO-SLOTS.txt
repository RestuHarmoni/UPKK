UPKK SmartKids - Audio Slot System

Cara guna:
1. Masukkan fail MP3 ke folder slot yang betul.
2. Pastikan nama fail sama tepat seperti senarai di bawah.
3. Zip/deploy semula app.
4. Clear cache atau buka versi baru app.

UI
- UI_CLICK              -> assets/audio/ui/click.mp3
- UI_TAB_SWITCH         -> assets/audio/ui/tab-switch.mp3
- UI_NOTIFICATION       -> assets/audio/ui/notification.mp3

QUIZ / LATIHAN
- QUIZ_CORRECT          -> assets/audio/quiz/correct.mp3
- QUIZ_WRONG            -> assets/audio/quiz/wrong.mp3
- QUIZ_ANSWER_SELECTED  -> assets/audio/quiz/answer-selected.mp3
- QUIZ_FINISH           -> assets/audio/quiz/quiz-finish.mp3

EXAM / PEPERIKSAAN
- EXAM_START            -> assets/audio/exam/exam-start.mp3
- EXAM_FINISH           -> assets/audio/exam/exam-finish.mp3
- EXAM_TIME_WARNING     -> assets/audio/exam/time-warning.mp3
- EXAM_SUBMIT           -> assets/audio/exam/submit.mp3

ACHIEVEMENT
- ACHIEVEMENT_UNLOCK    -> assets/audio/achievement/unlock.mp3
- STREAK_REWARD         -> assets/audio/achievement/streak.mp3

VOICE
- VOICE_CONGRATS        -> assets/audio/voice/tahniah.mp3
- VOICE_WRONG           -> assets/audio/voice/cuba-lagi.mp3
- VOICE_ENCOURAGE       -> assets/audio/voice/hebat.mp3
- VOICE_WELCOME         -> assets/audio/voice/selamat-datang.mp3
- VOICE_EXAM_FINISH     -> assets/audio/voice/exam-finish.mp3

Cadangan MP3:
- Format: .mp3
- Durasi: 2 hingga 8 saat
- Saiz: bawah 1MB setiap fail
- Nama fail: huruf kecil, tiada ruang kosong

Jangan guna Firebase Storage untuk audio jika mahu kekal mudah dan tanpa kos tambahan.
