Flow User:
1. User buka unlock/premium page.
2. Klik Bayar Sekarang RM49.90.
3. Browser buka ToyyibPay Bill Link.
4. Selepas bayar, user kembali ke app.
5. User isi optional Bill Code/Transaction ID.
6. User tekan Saya Sudah Bayar.
7. Rekod pending masuk Admin > Pembayaran ToyyibPay.

Flow Admin:
1. Admin buka Pembayaran ToyyibPay.
2. Semak bayaran sebenar di dashboard ToyyibPay.
3. Tekan Luluskan 1 Klik.
4. Sistem aktifkan subscription dan examLicense 365 hari.
